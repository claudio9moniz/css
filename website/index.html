Quiz

o que é HTML?
Trata-se de uma linguagem de marcação para desenvolver páginas e documentos eletrônicos para a internet , fornecendo informações para usuários, navegadores e mecanismos de busca
o que e css?
CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML.
o que e java script?
vaScript é uma linguagem de programação de alto nível criada, a princípio, para ser executada em navegadores e manipular comportamentos de páginas web.
